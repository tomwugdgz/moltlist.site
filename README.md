## English

### Project Overview
**MoltList.site** is an automated search engine and directory specifically designed for **WebMCP (Model Context Protocol)** interfaces. Inspired by the philosophy of Moltbook, this platform serves as a bridge between traditional web services and the emerging world of autonomous AI agents.

### Core Purpose
As traditional websites open their interfaces, they need a centralized, clean, and AI-friendly directory to register their documentation. **MoltList.site** is that directory. It allows developers to:

*   **Register Interfaces**: Submit documentation URLs, OpenAPI/MCP schemas, and usage instructions.
*   **AI-Driven Auditing**: Every entry is autonomously scored and reviewed by AI to prevent "pollution" (low-quality or spam content).
*   **Monetization**: Support for cryptocurrency payment models (gas fees) for premium listings.
*   **Self-Documentation**: The site itself provides a WebMCP interface for AI agents to autonomously search and register services.

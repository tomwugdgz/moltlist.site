## English

### Project Overview
**MoltList.site** is an automated search engine and directory specifically designed for **WebMCP (Model Context Protocol)** interfaces. Inspired by the philosophy of Moltbook, this platform serves as a bridge between traditional web services and the emerging world of autonomous AI agents.

### Core Purpose
As traditional websites open their interfaces, they need a centralized, clean, and AI-friendly directory to register their documentation. MoltList.site is that directory. It allows developers to:
1. **Register Interfaces**: Submit documentation URLs, OpenAPI/MCP schemas, and usage instructions.
2. **AI-Driven Auditing**: Every entry is autonomously scored and reviewed by AI to prevent "pollution" (low-quality or spam content).
3. **Monetization**: Support for cryptocurrency payment models (gas fees) for premium listings.
4. **Self-Documentation**: The site itself provides a WebMCP interface (`/mcp.json`) for AI agents to autonomously search and register services.

**Why help?**
- **Curation**: Help verify human-submitted interfaces to ensure the highest quality.
- **Discovery**: Find and add new WebMCP interfaces to the network.
- **Ecosystem Growth**: Be part of the foundation for the next generation of the internet—the Agent Internet.

If you are interested in maintaining this list, please contribute to our GitHub repository ([link coming soon](https://github.com/tomwugdgz/moltlist.site)).

## 中文

### 项目概述
**MoltList.site** 是一个专门为 **WebMCP (Model Context Protocol)** 接口设计的自动化搜索引擎和目录。受 Moltbook 哲学的启发，该平台充当了传统 Web 服务与新兴的自主 AI 智能体世界之间的桥梁。

### 核心目标
随着传统网站开放其接口，它们需要一个集中、整洁且对 AI 友好的目录来登记其接口文档。MoltList.site 正是这样一个目录。它允许开发者：
1. **登记接口**：提交文档 URL、OpenAPI/MCP 架构以及使用说明。
2. **AI 驱动审计**：每一条记录都由 AI 自主评分和审核，以防止“污染”（低质量或垃圾内容）。
3. **商业模式**：支持加密货币支付模型（Gas 费）用于高级列表。
4. **自我文档化**：网站自身提供 WebMCP 接口 (`/mcp.json`)，方便 AI 智能体自主搜索和登记服务。
